# การบ้าน HackerRank Software Testing 🚀

โปรเจกต์นี้คือการบ้านวิชา Software Testing ที่นำโจทย์อัลกอริทึมจากเว็บไซต์ HackerRank จำนวน 5 ข้อ มาเขียนโค้ดแก้ปัญหา พร้อมกับเขียนโค้ดทดสอบ (Unit Test) ให้ครอบคลุมการทำงานทุกบรรทัด (100% Code Coverage) 

## 📁 โครงสร้างโปรเจกต์ (Project Structure)
เพื่อให้การทำงานเป็นระเบียบและเป็นไปตามหลักปฏิบัติที่ดี (Best Practices) เราจึงแยก "โค้ดหลัก" กับ "โค้ดเทสอบ" ออกจากกันอย่างชัดเจนครับ:

*   **📂 โฟลเดอร์ `src/` (Production Code):** เก็บโค้ดสำหรับตอบโจทย์ทั้ง 5 ข้อ
    *   `funny_string.py`
    *   `alternating_characters.py`
    *   `caesar_cipher.py`
    *   `two_characters.py`
    *   `grid_challenge.py`
    *   `api_helpers.py` *(ฟังก์ชันพิเศษสำหรับทำ Test Stub)*
*   **📂 โฟลเดอร์ `tests/` (Test Code):** เก็บโค้ดทดสอบรันเงื่อนไขและ Edge cases แบบแยกข้อ
    *   มีไฟล์ขึ้นต้นด้วย `test_...` ตรงกับโจทย์ทุกข้อ
    *   มีไฟล์ `test_stub_bonus.py` *(สำหรับแสดงวิธีทำคะแนนพิเศษ)*

## ✨ สิ่งที่อยู่ในโปรเจกต์นี้
- **✅ โค้ดเทสครอบคลุม 100%:** ทดสอบมาอย่างหลากหลายกรณี (ทั้งกรณีปกติ ใส่ข้อความเข้ามารูปแบบแปลกๆ หรือกรณีสั้นสุดๆ) มั่นใจได้ว่าไม่มี Defect แฝงอยู่แน่นอน
- **⭐ คะแนนพิเศษ (Test Stub):** มีการนำเครื่องมือ `unittest.mock.patch` มาใช้เพื่อจำลอง (Stub) การเรียกข้อมูลแบบหลอกๆ (ในไฟล์ `test_stub_bonus.py`) ซึ่งตรงนี้ทำเพิ่มมาเพื่อรับคะแนนโบนัสตามกติกาของอาจารย์เลยครับ!

## 🛠️ วิธีรันเพื่อดูผลทดสอบและความครอบคลุม (100% Coverage)
ถ้าอาจารย์หรือใครก็ตามโหลดโฟลเดอร์นี้ไป สามารถรันเทสได้ง่ายๆ ตามขั้นตอนนี้ครับ (แนะนำให้รันผ่าน Command line ของ Windows):

1. **เปิดใช้งานสภาพแวดล้อม (Virtual Environment):**
   ```bash
   py -m venv venv
   .\venv\Scripts\Activate.ps1
   ```

2. **ติดตั้งเครื่องมือที่ใช้ทดสอบ:**
   ```bash
   py -m pip install pytest pytest-cov mock
   ```

3. **รันคำสั่งทดสอบดูผลลัพธ์ 100%:**
   ```bash
   py -m pytest tests/ --cov=src --cov-report=term-missing
   ```
   👉 *หลังจากรันคำสั่งนี้แล้ว โปรแกรมจะจัดการเทสทุกเงื่อนไข ถ้าสำเร็จมันจะขึ้นตัวหนังสือสีเขียวว่า `PASSED` และในตารางบรรทัดล่างสุด (TOTAL) ช่อง Cover จะขึ้นเป็น `100%` ตามที่อาจารย์ระบุไว้ครับ*
